  1. Option 1: When, you commit, use git commit's -s flag

    git add changed_file.go changed_file_test.go
git commit -s

    Make sure you have run:

    git config --global user.name "Your Name"
git config --global user.email youremail@domain.com

    If you need to revise existing commits to add the DCO, see this excellent reference on rewriting git history. Once you have revised the commits, you can use git push -f on the branch which you are using for the pull request. Be careful, however, not to accidentally over-write existing work.

  2. Option 2: Fill in your credentials and copy-paste from the box that appears at the bottom of the page:

    Docker-DCO-1.1-Signed-off-by: {{name}} <{{email}}> (github: {{github}})
